Terminal/CMD commands

Plebnet Wiki
https://plebnet.wiki/wiki/Main_Page



The following Citadel commands can also be used for Umbrel users. Just replace citadel with umbrel.

Connect to your Node:
ssh YourIP@Poort
(for example: ssh 234.0.0.0@5500 or ssh citadel@citadel)

List sub folders:
dir

Move to a folder/file:
cd ReplaceThisWithTheFolderName

Move a folder back:
cd ..

Edit a File
nano ReplaceThisWithTheFileName
(for example: nano lnd/lnd.conf)

Add a Watchtower (LND, Dockerbased env. like Citadel/ Umbrel)
docker exec -it lnd lncli wtclient add

List added Watchtowers (LND, Dockerbased env. like Citadel/ Umbrel)
docker exec -it lnd lncli wtclient towers

Redo update after failiure (Citadel)
sudo rm -rf ~/citadel/statuses/update-in-progress && sudo ~/citadel/scripts/update/update --repo runcitadel/core#stable

Stuck loading screen after succesfull update(Citadel)
sudo rm ~/citadel/nginx/nginx.conf && sudo docker kill nginx && sudo ~/citadel/scripts/start
